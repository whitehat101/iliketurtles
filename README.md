# iliketurtles
ComputerCraft APIs for turtle lovers!

## Install

Requirements:
* http must be enabled
* raw.github.com must be whitelisted

```sh
# openp's github script
github run whitehat101/iliketurtles/master/install.lua
```
or

```sh
# any turtle
pastebin get sERDBEru install
install
rm install # optional
```
